== Installation ==

1. Upload plugin folder `translate-content` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setup your ad widget through the 'Themes/Widgets' menu in WordPress

== Frequently Asked Questions ==

= How can i get API key? =
Go to https://code.google.com/apis/console
Create project, select Translate API, accept pricing and attach wallet.
Then in API Access, you can see your API key.
